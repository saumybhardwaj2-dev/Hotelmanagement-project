# Hotel Management Website Project

Yeh ek basic Hotel Management System ka web project hai jisme multiple destinations aur booking forms provide kiye gaye hain.

## Features
* **Home Page**: Alert message aur navigation links ke saath yellow theme.
* **Destinations**: Brazil, Hong Kong, aur Thailand ke liye dedicated information pages.
* **Special Rates**: AAA, AARP, aur Military discounts apply karne ke liye form.
* **Luxury Stays**: Homes & Villas section premium rentals ke liye.
* **Free Offers**: Free night awards aur alternative lodging ki jankari.

## Project Structure
* `hotelmanagementwebsite.html` - Main Landing Page
* `hotels.html` - Destination Directory
* `desti.html` - User Inquiry Form
* `offers (1).html` - Special Rates Form
* `travellingexperience.html` - Luxury Rentals Page

## Cultural Insights
Project mein Brazil jaise deshon ke local hygiene customs (jaise frequent showering) ko bhi highlight kiya gaya hai.

## Tech Used
* HTML5
* CSS (Inline styling)
* JavaScript (Basic Alerts)
